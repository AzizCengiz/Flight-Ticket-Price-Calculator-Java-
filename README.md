✈️ Java ile Uçak Bileti Fiyatı Hesaplama
📌 Proje Hakkında

Bu proje, Java programlama dili kullanılarak geliştirilmiş basit bir konsol uygulamasıdır. Uygulama, kullanıcının girdiği bilgilere göre uçak bileti fiyatını hesaplar.

Geliştirme ortamı olarak Eclipse IDE kullanılmıştır.

🛠 Kullanılan Teknolojiler

Java

Eclipse IDE

12–24 yaş arası → %10 indirim

65 yaş üstü → %30 indirim
